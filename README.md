# RS3 Money Advisor (preview)

RuneScape 3 companion to the [OSRS Money Advisor](https://github.com/Yousdje/osrs-money-advisor)
— currently a **preview build**: the layout and styling are final, but the rows
are illustrative samples (clearly badged) while the live RS3 data pipeline is
under construction.

The plan mirrors the OSRS site: a self-hosted advisor generates `data.json`
from live RS3 price data with conservative conventions (real spreads, volume
caps, verified money-maker rates), pushed automatically every few hours.
